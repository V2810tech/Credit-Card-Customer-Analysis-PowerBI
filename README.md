# Credit Card Customer Analysis - Power BI

## 📌 Project Overview
This project presents an interactive Power BI dashboard built using Credit Card Customer data. The dashboard provides meaningful insights into customer demographics, transaction behavior, spending patterns, and financial performance through interactive visualizations.

---

## 📊 Objectives
- Analyze customer demographics.
- Monitor transaction and spending trends.
- Compare customer segments.
- Build interactive dashboards with filters and slicers.
- Generate business insights for decision-making.

---

## 🛠️ Tools Used
- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel (Dataset)
